# Rust Url Shortener
> This is a Rust study project for the purpose of knowledge ONLY.
![](demo.mov)
## Dependencies
- tokio
- warp
- redis
- percent-encoding 
## Short Url Generator
```
$ cd short-url-generator
$ cargo run
```
Open browser in:
```
http://localhost:8000
```
## Shortener Redirect
```
$ cd shortener-redirect
$ cargo run
```
